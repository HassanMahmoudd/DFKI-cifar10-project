DFKI Test Project - Image Classification

Description:

- This project is a test project for DFKI HIWI position on Applied Machine Learning.
- It's an image classification application trained on cifar10 popular dataset.
- This app can classify any image into one of 10 categories as follows:-
'Airplane', 'Automobile', 'Bird', 'Cat', 'Deer', 'Dog', 'Frog', 'Horse', 'Ship', 'Truck'
